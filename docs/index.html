01 · O que é

Os melhores modelos de imagem, sem sair da conversa

Um servidor MCP local que dá ao Claude Code o poder de criar, editar e preparar imagens — feito para quem não programa.

Converse, não configure

"Gere uma imagem de um escritório moderno, 16:9" — é só isso. O resultado cai pronto na pasta Imagens/image-mcp.

Dois provedores, um lugar

GPT Image 2 (OpenAI) para composições complexas e texto na imagem; Nano Banana (Google) para volume rápido e barato — escolha por pedido.

Edição com referência

Parta de fotos existentes: troque o fundo, combine elementos de várias imagens, gere variações de produto — com máscara opcional nos modelos OpenAI.

Acabamento grátis

Converter formato, redimensionar, comprimir e gerar link público da imagem — tudo local ou em serviço gratuito, sem gastar um centavo de API.

02 · Como funciona

Do pedido em português ao PNG na pasta

O servidor roda na sua máquina (via npx, sem instalar nada manualmente) e fala direto com o provedor usando a sua chave — sem intermediário.

1

Você pede em português

"Gere uma imagem de produto de uma caneca azul, fundo branco". Se você não disser o formato, o Claude pergunta antes — oferecendo os padrões de mercado (1:1, 9:16, 16:9…).

2

O Claude escolhe a ferramenta certa

generate_image para criar do zero; edit_image sempre que existe imagem de partida — referência, combinação de fotos ou máscara marcando a região a alterar.

3

O servidor chama o provedor com a sua chave

A requisição vai direto da sua máquina para a API da OpenAI ou do Google (conforme o modelo pedido). A chave fica só na configuração local do Claude Code — nunca passa por servidor de terceiros.

4

A imagem volta e vira arquivo

O resultado chega em base64 e é salvo como PNG na pasta Imagens/image-mcp (configurável via IMAGE_MCP_OUTPUT_DIR), com nome datado e descritivo.

5

Acabamento opcional — sem custo de API

Converter/redimensionar/comprimir localmente (convert_image) e, quando a plataforma pede o link da imagem, hospedar de graça com host_image — permanente ou expirando sozinho em 1h–72h.

03 · Ferramentas

Seis ferramentas, quatro delas grátis

Só a geração e a edição consomem créditos do provedor — o resto roda local na sua máquina ou em serviço gratuito.

generate_image

Cria imagem(ns) a partir de um texto, em qualquer um dos 6 modelos registrados, e salva em disco. Até 4 por pedido.

usa API do provedor
edit_image

Edita ou gera a partir de imagens de referência: troca de fundo, combinação de fotos, variações. Máscara (região exata) nos modelos OpenAI.

usa API do provedor
convert_image

Converte png/jpeg/webp/avif, redimensiona e comprime — processamento local com sharp, instantâneo.

grátis · local
get_image_info

Dimensões, formato, peso em disco e transparência de qualquer imagem local — útil antes de editar ou publicar.

grátis · local
host_image

Sobe a imagem para hospedagem gratuita e devolve a URL pública — permanente ou temporária (1h a 72h). Sem cadastro, sem chave.

grátis · sem cadastro
list_image_models

Lista os modelos disponíveis no servidor e o que cada um suporta (edição, padrão).

04 · Modelos e custo

Transparência: quem cobra é o provedor

O Image MCP é gratuito e de código aberto. As imagens geradas consomem créditos da sua chave de API — na OpenAI ou no Google.

Cada imagem gerada tem custo de API

OpenAI cobra por imagem (varia com qualidade e tamanho) e exige créditos pré-pagos. O Google tem nível gratuito limitado — dá para começar sem pagar nada — e depois cobra por imagem. As ferramentas locais (convert_image, get_image_info) e o link público (host_image) não custam nada. Durante a instalação guiada, o Claude avisa esses custos antes de concluir.

Cole este prompt no Claude Code
Leia https://raw.githubusercontent.com/Expert-Integrado/image-mcp/main/SETUP.md e instale o servidor MCP de imagens da Expert Integrado para mim, seguindo o protocolo de onboarding assistido descrito lá: verifique os pré-requisitos, me pergunte com botões se eu quero que você faça as etapas de navegador por mim (criação das chaves de API), me avise dos custos por imagem, valide as chaves sem custo, registre o MCP e faça o teste final. Me peça o que precisar.

Etapas de navegador assistidas

Criar a chave no painel do provedor é no navegador — o Claude pergunta se você quer que ele faça por você (via Playwright MCP ou Claude in Chrome) ou prefere o passo a passo manual. O login é sempre você quem faz.

Validação sem custo

Antes de concluir, o Claude confere cada chave num endpoint gratuito do provedor (lista de modelos) — sem gerar imagem, sem gastar crédito.

Segredos no lugar certo

A chave entra direto na configuração local do Claude Code — nunca em repositório, chat público ou log. Trate chave de API como senha de banco.

Teste final com aviso

No fim, o Claude pede para reiniciar o Claude Code e gera uma imagem de teste — avisando antes que ela já consome crédito (ou o nível gratuito do Google).
